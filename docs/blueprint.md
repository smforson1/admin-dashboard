# **App Name**: SchoolWise Dashboard

## Core Features:

- Dashboard Overview: Display key metrics like student, teacher, and class counts, Paystack balance, SMS credits, and interactive charts for system activity and user logins.
- Student Management: Enable admins to add, edit, and view student profiles, assign students to classes and guardians.
- Teacher Management: Allow adding, editing, and assigning teachers to classes and subjects, and manage permissions for assessment editing.
- Class Management: Functionality to add and edit classes, create subclasses, and manage teacher and student assignments.
- Assessment Management: Creation, release, and locking of assessments; generating and printing assessment reports.
- Mock Exam Management: Create, manage, and end mock exams, and select eligible classes. Exams can be toggled between editable and non-editable states.
- Intelligent Notification Recommendations: Leverage AI to analyze historical data and suggest the optimal timing and content for notifications to different target groups, acting as a tool for the administrator to improve engagement and communication.
- Payment Tracking: Configure prices for mock exam and terminal report vouchers for web and mobile platforms; track all payments with history and transaction details.
- Notification System: Create, send, and delete notifications to specific target groups (students, teachers, parents, classes) with full history.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to convey trust, authority, and knowledge. This color provides a professional yet modern feel, essential for an educational platform.
- Background color: Light Lavender (#E6E6FA), a desaturated version of the primary, creating a calm and unobtrusive backdrop for content.
- Accent color: Violet (#8F00FF), an analogous hue to the primary color that offers a vibrant contrast for interactive elements and key actions. 
- Body and headline font: 'Inter', a sans-serif typeface known for its readability and modern design, suitable for both headlines and body text across the dashboard.
- Use a consistent set of icons from a library like FontAwesome or Material Icons, styled in a flat design with the accent color to maintain a clean and modern look.
- Employ a responsive, grid-based layout to ensure usability across different screen sizes. Prioritize clear hierarchy and logical grouping of information.
- Incorporate subtle animations for transitions and interactions to enhance user experience without being distracting, such as smooth loading indicators and interactive chart animations.
